About Me

工作上的專長是以 .NET Framework C# 相關技術為主；網頁部分從基本的 WebForm、MVC、WebAPI 到 SPA 都有玩過，視窗程式則是因要設計公司的自動化商務環境，而開始寫 WinForm，最近新的專案都已著手更新為 WPF 來設計；也因此有寫過不少配合 SAP 的外部應用程式。

網頁專業內的其他興趣則是 Front-end 相關領域；雖然目前的工作都不太要求前端的設計風格，但我個人都會去研究一些前端技術和UX，只要讓使用者能用起來順暢、舒服、不要有疑惑，就會覺得很滿足。早年喜歡使用 Bootstrap + jQuery 去設計前端頁面，近年偏好 Bootstrap + Vue.js，使用起來也很順手。

因從國中就開始幫人修電腦，在求學期間的打工主要在維護和管理電腦資訊系統，有點像 MIS 的工作。

專業外的興趣就是動漫遊戲了，固定會追一些質感不錯的新番；遊戲因近年開始工作，時間上只能玩簡單的手遊，主機類遊戲都沒在玩了，有點小小遺憾。

戶外活動上，近年喜歡規劃自助旅行，近至台灣深山、離島，遠至日本，都一手包辦旅遊行程；偏好了解當地文化和歷史故事的旅程。

如果有其他什麼想聊聊的，歡迎來互相認識一下哦！

我自身的期許與經歷

出社會工作也快五年了；看了看五年前自己寫的自傳後，決定用另外一種方式來詮釋自己。




「期望自己能透過資訊專長為他人、為社會有所幫助和貢獻。」

我試著將懂事至今，自己的工作動力濃縮為一句話。




大學畢業後，很幸運地，在高雄有間小型企業把我找了進去，主管的耐心和同事的指導下，從簡單的網頁程式到整個公司的後台系統重建，都漸漸玩出心得。

雖然在這過程公司重組轉型了數次，同事們來來去去，但因這些合作的機緣和他們成為了朋友，即使不在同個工作崗位也一直保持著聯繫。

而我也為了要跟上資訊圈子的變化，在前同事的推薦下，開始接觸一些資訊社群的活動，其中 GDGK、高雄前端社群、K.NET、KSDG 都是我常去參加的社群；將社群學到的技術經驗收納回公司的專案產品裡，再以此優化自己專案幫助使用者，希望讓別人知道，高雄也能產出很好的軟體使用體驗。



「說到頭來，工作不就是為了能持續幫助別人嗎？」



回想起求學階段，我必須承認我是個不愛讀書的孩子，不知讀書是為了要成就什麼，也不知讀了書能為自己得到什麼。

跟很多小朋友一樣，小學時就沉迷於電腦遊戲，小六開始戴眼鏡；正是這般沉迷，使我很小就開始研究起電腦作業系統。我開始為同齡小朋友研發遊戲外掛，漸漸地，連電腦的其他問題也開始跑來詢問我，舉凡中毒、驅動程式問題、災難性錯誤......等等，在那個Google也不太好用的年代，開始學著上網把問題找出來並且解決。


這陣風也隨著我升上國中，班導師直接指名我來設計班級網站，甚至在國三時推薦我為技優生；國中是我開始研究電腦硬體的時期，人生第一次自己組裝電腦也是在這時，越來越多同學找我開「電腦零組件估價單」，沒日沒夜地無償幫助同學組裝電腦、安裝作業系統，也認識了很多志同道合的動漫好朋友（題外話）。

除此之外也參與了證照考試、技職比賽，最後也是在技優推舉下，進入了國立東勢高工的資訊科。


不過不愛讀書的「技術債」也在升上國立高職後開始償還了；即使術科在班上都名列前茅、證照高分過關，但學科卻是每一學期都吊車尾，不管怎麼熬夜讀書、不管怎麼上課抄筆記，總是沒辦法追上同學。

「我拉低了班上的平均分數」、「我又讓老師教到很灰心了」、「這科又被當了，暑假又要來重補修了」心裡頭總是有一種聲音在責備不中用的自己，使我不敢面對人群，埋頭都在用錯誤的方式讀書。還好同學和老師願意體諒我，願意教我功課、願意多給我一點時間。



然而在準備升大學時，我做了一個要離開家鄉的決定。



講到這裡需要提一下我的成長背景。

我的老家在台中市區，父母從小離異，母親從我小學畢業後就再也沒見過，父親則很少在家、也很少談話，主要由阿公阿嬤帶大。而阿公阿嬤在我心中，相當於真正的父母，對我影響深遠。他們分別出生在民國十到二十年左右，也就是日治時代後期；經歷了戰亂、國民政府遷台、解嚴，想必是很有故事的人。只是可能也是因為這樣的經歷，他們不敢多講，我也來不及聽他們述說。阿嬤在我小學時過世、阿公到了我大學時也走了，享耆壽95歲。


說回當時升大學時，選了高雄和台北的學校。

原因無他，只是因為我父親在我國高中那幾年胡亂揮霍，散盡阿公的存款；勸阻無用又脾氣暴躁，整個家只要他在就烏煙瘴氣。

我感覺我留在這個家只會被不好的氛圍影響，所以便在上高中的那一刻就決定以後要上台北或高雄的學校，在外縣市學著獨立生活幾年，等到有辦法獨立工作時再回來接阿公。


這麼想著的我，最後分發上了樹德科技大學的資訊管理系。

不過事與願違，阿公在我大三那一年離世了。來不及孝順阿公阿嬤的這份遺憾，讓我更加珍視身邊的人、重視被他們養大的自己的生活。


在大學期間我受到導師重用，沒多久後就開始幫系上管理電腦教室、維護電腦設備，升上二年級更被推舉為班級代表，處理班務和系上各種電腦資訊相關的問題。

大三當上系上的程式設計助教，主要指導JAVA和C#的基本概論。

在課餘時間則是在經營遊戲私人伺服器，規劃遊戲內容與策劃活動。當時較活躍於「當個創世神（Minecraft）」的遊戲管理上，可以至Google搜尋「種子伺服器」看到我設立的簡易官網。


就算如此，我在大學四年級關鍵的畢業實務專題上卻面臨了各種挫折，幾年來的獨立處事碰上團隊溝通出現了許多問題；我曾經因此低落、無力、喪氣，體悟到了很多東西並不是想像中那麼好掌控，漸漸地開始學會該怎麼融入團體、該怎麼換位思考，在沮喪中找到前進的動力。

最後我們的專題成果雖然不甚理想，但我們從中學會了很多，也體會到業界APP程式設計的辛苦。明白能力往往只是一個基本，沒有良好的溝通能力將無法讓團隊有所前進；這是我的專題小組、專題老師在畢業前給我的最大禮物。


除了學業以外，我帶著小時阿嬤告訴我台灣的美景的期待，跟兩個朋友一起挑戰了機車環島。總共騎了十一天回到高雄，這趟旅行使我打開了新的視野還有對於旅行的體悟。

後來這幾年，自己安排了無數旅行，近至綠島蘭嶼流浪，遠至日本自助旅行；從怕生的自己，漸漸變得能接受人群、接受不同的文化思維。

跳脫原本讀書的方法，從其他角度去理解，甚至在學了日文後，發現改進自己學習英文的方式。



「原來不是做不到，而是用的方法不適合自己而已。」



在資訊這個小小圈子走跳，除了不斷學習、精進自己以外，對於其他領域的視野、做人處事的態度，才是身為人應該要成長的地方。

畢業後我選擇留在高雄為在地盡一份力，冀望有朝一日能讓在高雄成長的人們有更多資訊工作機會。

我沒有很聰明的腦袋、英文也沒有很好，即便至今也還在摸索讀書的方式。

但我喜歡工作中能為他人貢獻的價值，我明白有多少人能透過資訊軟體整合，省下多少寶貴時間去陪伴自己的家人、創造自己的愛好。

期許自己身體可以健康，能不斷精進；也期待有相同志向的公司願意接納這樣懵懂的自己。



這就是我


「期望自己能透過資訊專長為他人、為社會有所幫助和貢獻。」的我。
